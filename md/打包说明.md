1.pyinstaller
pip install pyinstaller
pyinstaller -F -w --upx-dir upx --optimize 2 image_compressor.py

参数说明
参数	作用
-F	打包成单个 exe 文件
-w	GUI 程序模式，隐藏黑框窗口
--upx-dir upx	使用 UPX 压缩工具（需已安装并加入 PATH）
--optimize 2	优化级别为 2（最高级别）
image_compressor.py	你的主脚本文件名

2.Nuitka
Windows → 默认使用Clang, 也可以用 Visual Studio Build Tools 或 MinGW
Linux/macOS → 使用系统自带的 gcc/clang

pip install nuitka ordered-set zstandard

nuitka ^
  --onefile ^
  --lto=yes ^
  --windows-console-mode=disable ^
  --output-dir=dist ^
  --enable-plugin=tk-inter ^
  image_compressor.py

upx --best --lzma dist\image_compressor.exe
upx --ultra-brute dist\image_compressor.exe

参数说明
参数	作用
--onefile	打包成单个 exe 文件
--clang	使用 Clang 编译器（需已安装并加入 PATH）
--lto=yes	启用链接时优化（Link Time Optimization）
--nofollow-imports	只编译用到的模块，忽略未使用的 import
--no-pyi-file	不生成类型提示文件
--windows-disable-console	GUI 程序模式，隐藏黑框窗口
--output-dir=dist	输出到 dist 文件夹
image_compressor.py	你的主脚本文件名
